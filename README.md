# Rand Card #

随机抽取游戏王卡片，并生成ydk文件

## 🚀 快速开始 ##

1. 打开bash，输入

```
git clone https://github.com/MRoldL001/rand_card
```

这一步会将项目完整克隆到本地

2. 在 `./data_src/` 下创建文件夹 `img` ，然后将需要抽取的卡片存入 `./data_src/img` 

3. 运行 `./data_src/双击我将卡片写入数据库.bat` 来将数据写入 `./data/cardNum.db` 

4. 运行 `./双击我开始卡片抽取.bat` ，输入抽取数量，开始卡片抽取
